## Dev

1. Clonar el repositorio

2. Instalar módulos de node

```
npm i
```

3. Cambiar las variables de entorno en el archivo .env.template y renombrarlo a .env

4. Base de datos

```
docker compose up -d
```

5. Para ejecutar el servidor

```
npm run dev
```
